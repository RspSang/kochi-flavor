# Kochi Flavor

> 新しい方式で高知県のグルメ情報を紹介し共有、記録することができるウェブアプリケーション

> 색다른 방식으로 일본 고치현의 맛집을 소개하고 공유,기록할수있는 웹어플리케이션

# 프로젝트 소개 プロジェクトの紹介
## 1.만든이유 作ったきっかけ

내가사는 고치현은 일본에서도 작은 소도시에 속하지만 음식 하나만큼은 이지역에서만 나오는 특산물을 이용해 만든 매력있고 맛있는 음식이 많아 전국에서도 손꼽히는곳이다

하지만 이런 매력넘치는 고치현이 지방이라는 이유로 도심특화의 미디어,어플등 다양한 서비스에 도태되어 아직도 맛집 정보를 찾으려고하면 인터넷 검색이나 sns 뉴스,잡지등의 제한된 미디어 정보가 전부였다

물론 구글맵이나 인스타 게시물등 도심과 비슷한 서비스 방식들이 있었지만 고치현 특화의 새로운 방식으로 정보전달을 할수있는 어플을 만들어 제공하고싶은 마음에 어플개발에 도전하게 되었다

私が住んでいる高知県は小さい都市ですが、食べ物だけは高知ならではの特産品を利用して作った魅力ある美味しい食べ物がたくさんある全国でも有名な県です。

しかし、こんなに魅力あふれる高知県が地方という理由で都心特化のメディア、アプリなど様々なサービスに離されグルメ情報を探そうとしても未だにインターネット検索や sns、ニュース、雑誌などの限られたメディア情報が全てでした。
もちろんグーグルマップやインスタ投稿など都心と似たサービスはありましたが、高知県特化の新しい方法で情報伝達ができるアプリを作って提供したいと思い、アプリ開発に挑戦しました。
